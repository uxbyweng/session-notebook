# Session Notebook

Dieses Session Notebook dokumentiert alle Lerninhalte meines Web Development Bootcamps bei Spiced Academy (Nov 2025 – März 2026).  
Jeder Tag wird als eigene Markdown-Datei festgehalten und nach Themenbereichen sortiert (HTML, CSS, JavaScript, Shell, Git, etc.).

Das Ziel dieses Notebooks:

-   Lernfortschritt transparent festhalten
-   Inhalte schnell wiederfinden
-   Übungen, Challenges und Code-Beispiele zentral gesammelt haben
-   Eine saubere Grundlage für Wiederholung, Prüfungsvorbereitung und spätere Projekte schaffen

Alle Notizen werden ausschließlich im Terminal angelegt und gepflegt, wie im Bootcamp gefordert.
